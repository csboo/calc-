# calcplusplus
A very simple command line tool to solve basic math expressions, written in c++

# how to build
g++ -o calc++ ./calc.cpp

# how to use
type 
~ calc++ 
to get a brief help message

~ calc++ "expression"
where 'expression' can be one of many mathematical expressions.

# supported currently
- addition "+"
- subtraction "-"
- multiplication "*"
- division "/"
- exponentiation (power) "^"
- parentheses
